Breakdown of the calculation
xG formula explanation
xG = ML(features)

The trained model estimates goal probability from shot location, distance, angle, body part, goalkeeper position, defenders, attackers, zone, marking, wall size, and delivery context.

set-piece value = P(shot) x xG

For direct free kicks, P(shot) = 1.00 because the free kick is already the shot.

PSxG formula explanation
speed_mps = shot_speed / 3.6
ball_time = shot_distance / speed_mps
GK_dive = sqrt(dx^2 + dy^2)
GK_reach = reaction + max(dx / GK_H, dy / GK_V)
margin = GK_reach - ball_time

PSxG rises when the ball reaches a difficult target faster than the goalkeeper can react and cover the horizontal or vertical distance.
